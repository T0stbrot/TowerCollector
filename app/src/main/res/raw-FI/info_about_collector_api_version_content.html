Voit päättää, mitä menetelmää käytetään matkapuhelinmaston tietojen saamiseksi mittauksia kerättäessä.
On erittäin suositeltavaa käyttää uusinta saatavilla olevaa menetelmää luotettavampien tietojen saamiseksi. Valinta ei ole käytössä laitteissa, joissa on Android 4.1 tai aikaisempi.

  • Automaattinen tunnistus – Sovellus tunnistaa automaattisesti, mitä API-versiota voidaan käyttää ja valitsee parhaan vaihtoehdon. Se käyttää turvallista vaihtoehtoa jos Collector käynnistyy ilman signaalia.
  • Android 4.2 API - Käyttää Android 4.2:n ja uudempien versioiden käytettävissä olevia menetelmiä. Nämä menetelmät tarjoavat luotettavimmat solutiedot. Jotkin valmistajat eivät kuitenkaan ole toteuttaneet uutta API:a oikein, eikä sitä ehkä voi käyttää tällaisissa laitteissa. Android 5.1:ssä ja uudemmissa versioissa voidaan palauttaa useiden SIM-korttien matkapuhelinmastot.
  • Android 1.0 API - Käyttää menetelmiä, jotka ovat saatavilla Androidin ensimmäisestä versiosta lähtien. Nämä menetelmät voivat palauttaa epäluotettavaa dataa LTE ja UMTS -verkoissa. Laitteesta riippuen tämä menetelmä saattaa edellyttää näytön päällä pitämistä keräämisen aikana, jotta solujen tiedot päivittyvät oikein.
