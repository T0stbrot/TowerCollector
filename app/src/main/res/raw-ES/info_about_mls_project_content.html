BeaconDB es una base de datos pública que recopila información de ubicación sobre infraestructura inalámbrica común (torres de telefonía móvil, puntos de acceso WiFi, balizas Bluetooth). Los datos de las torres de telefonía celular que aporte se agregarán y se harán públicos bajo la licencia "Creative Commons (CC-0)".
Tower Collector le brinda la oportunidad de contribuir en el proyecto BeaconDB cargando torres de telefonía celular de su área. Más información.
